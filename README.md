# Stump CLI

Initialize a WordPress project based on the [Stump](https://github.com/kohactive/stump) theme

## Installation

`$ gem install stump-cli`

## Usage

`stump-cli init NAME`  
_See the [Stump repo](https://github.com/kohactive/stump) for theme usage_
